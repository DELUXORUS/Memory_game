Memory Game (TCP Client-Server)

Консольное приложение «Игра на запоминание» (Memory Game), реализованное на C++ с использованием Berkeley Sockets. Приложение представляет собой клиент-серверную систему для тренировки памяти с поддержкой режима многопользовательской игры.
1. Описание проекта

Игра заключается в последовательном воспроизведении символов, которые генерирует сервер. В каждом раунде последовательность удлиняется.

    Сервер: однопроцессное однопоточное приложение, использующее неблокирующие сокеты и системный вызов poll(2) для обеспечения высокой производительности и одновременной работы с неограниченным числом клиентов.

    Клиент: консольное приложение с визуализацией игрового процесса, поддерживающее как интерактивный режим, так и режим тестирования производительности.

2. Сетевой протокол

Приложение использует разработанный протокол прикладного уровня поверх TCP. Все сообщения передаются в текстовом виде, а разделителем является символ переноса строки (\n).
Команды клиента

    начать / start — инициализация новой игровой партии.

    рекорд / record — запрос текущего личного рекорда.

    сдаюсь / give_up — досрочное завершение партии.

    тест / test — режим стресс-тестирования производительности (без проверок).

Ответы сервера

    welcome — подтверждение подключения.

    record:<значение> — текущий рекорд игрока.

    symbol:<символ> — новый символ для запоминания.

    correct — верный ввод последовательности.

    wrong:<длина>:<рекорд> — сообщение об ошибке с итоговой статистикой.

3. Технические особенности

    Реализация: C++, Berkeley Sockets API.

    Архитектура: Event-driven на основе poll(2).

    Неблокирующий ввод-вывод: Сокеты переведены в O_NONBLOCK для предотвращения блокировок сервера.

    Производительность: Однопоточная обработка эффективно использует ресурсы CPU без накладных расходов на переключение контекста.

4. Сборка и запуск
Требования

    Компилятор g++ (стандарт C++17 или выше).

    ОС: Linux / Unix-like.

Компиляция
Bash

# Клонируйте репозиторий
git clone <url-вашего-репозитория>
cd memory-game

# Сборка проекта
make

Запуск

Сервер:
Bash

./server <порт>

Клиент:
Bash

./client <имя_игрока> <ip_сервера> <порт>
